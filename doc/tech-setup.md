# 技术搭建指南

## 方案概述

本平台采用「零代码组合方案」，核心工具：
- **Notion + Super.so**：前台内容展示
- **飞书**：互动系统和数据管理
- **微信公众号**：用户沉淀

**预估成本**：年费约1000元

---

## 第一部分：Notion内容站点搭建

### 1.1 注册Notion账号

1. 访问 [notion.so](https://notion.so)
2. 点击「Sign Up」注册账号
3. 建议使用常用邮箱，方便后续管理
4. 选择「Personal」个人版（免费）

### 1.2 创建工作区

1. 登录后，点击左侧「+」创建新工作区
2. 工作区名称：`专业真相`
3. 选择「Personal」个人版

### 1.3 搭建页面结构

在根页面创建以下结构：

```
📄 专业真相（首页）
├── 📄 关于我们
├── 📄 专业百科
│   ├── 📄 工学类
│   │   ├── 📄 计算机科学与技术
│   │   ├── 📄 电子信息工程
│   │   └── 📄 ...
│   ├── 📄 经济学类
│   ├── 📄 管理学类
│   └── 📄 ...
├── 📄 问答广场
├── 📄 专家入驻
└── 📄 联系我们
```

### 1.4 制作首页

**首页内容建议**：

```markdown
# 专业真相

> 高考选专业，先看真相

## 我们是谁

一群来自各行各业的过来人，愿意无偿分享真实的专业经历，
帮助学弟学妹们做出更适合自己的选择。

## 我们有什么不同

❌ 不是付费咨询
❌ 不推销任何机构
❌ 不只有一家之言

✅ 完全免费
✅ 多元视角
✅ 真实经历

## 快速入口

[📘 专业百科](/专业百科) - 浏览各专业真实信息
[💬 我要提问](飞书表单链接) - 提出你的问题
[👨‍🏫 成为专家](/专家入驻) - 分享你的经验

## 联系我们

- 公众号：【待填写】
- 邮箱：【待填写】
```

### 1.5 制作专业百科页面

参考 `content-template.md` 模板，为每个专业创建详情页。

**技巧**：
- 使用Notion的Database功能批量管理专业
- 每个专业可以是一个Page
- 使用模板功能快速创建新专业页面

### 1.6 绑定自定义域名（可选但推荐）

**使用 Super.so**：

1. 访问 [super.so](https://super.so)
2. 注册账号
3. 连接Notion页面：
   - 点击「Add Site」
   - 授权Notion访问
   - 选择「专业真相」工作区
4. 配置域名：
   - 使用Super提供的免费子域名（如：专业真相.super.site）
   - 或绑定自己的域名（需$12/月）
5. 自定义样式：
   - 上传Logo
   - 设置主题色
   - 添加导航菜单

**绑定自己的域名**：

1. 购买域名（阿里云/腾讯云，约60元/年）
2. 在Super后台添加域名
3. 在域名服务商添加DNS解析：
   - 类型：CNAME
   - 名称：@
   - 值：Super提供的地址
4. 等待DNS生效（10分钟-24小时）

## 第二部分：飞书互动系统搭建

### 2.1 注册飞书账号

1. 访问 [飞书](https://www.feishu.cn)
2. 点击「免费使用」注册
3. 创建组织：`专业真相`
4. 完成组织认证（可选）

### 2.2 创建多维表格

#### 表1：专家信息表

1. 新建多维表格，命名为「专家信息表」
2. 添加字段：

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 姓名 | 文本 | 专家真实姓名 |
| 昵称 | 文本 | 对外展示昵称 |
| 认证状态 | 单选 | 待审核/已认证/已拒绝 |
| 擅长专业 | 多选 | 可选择的专业列表 |
| 当前公司 | 文本 | |
| 职位 | 文本 | |
| 工作年限 | 数字 | |
| 毕业院校 | 文本 | |
| 所学专业 | 文本 | |
| 学历 | 单选 | 本科/硕士/博士 |
| 个人简介 | 文本 | |
| 联系方式 | 文本 | 手机或微信 |
| 认证材料 | 附件 | 图片/PDF |
| 认证等级 | 单选 | 铜牌/银牌/金牌/钻石 |
| 创建时间 | 创建时间 | 自动 |
| 审核备注 | 文本 | 审核说明 |

#### 表2：问答管理表

1. 新建多维表格，命名为「问答管理表」
2. 添加字段：

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 问题ID | 自动编号 | |
| 问题内容 | 文本 | 学生提问 |
| 提问者姓名 | 文本 | |
| 提问者联系方式 | 文本 | 可选 |
| 问题分类 | 单选 | 专业选择/就业前景/学习内容/院校选择/其他 |
| 相关专业 | 多选 | |
| 问题状态 | 单选 | 待回答/已回答/已关闭 |
| 指派专家 | 人员 | 或关联专家表 |
| 回答内容 | 文本 | |
| 回答专家 | 文本 | |
| 回答时间 | 日期 | |
| 浏览量 | 数字 | |
| 点赞数 | 数字 | |
| 收藏数 | 数字 | |
| 是否公开 | 复选框 | 默认公开 |
| 创建时间 | 创建时间 | |

#### 表3：专业百科内容表

1. 新建多维表格，命名为「专业内容表」
2. 添加字段：

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 专业名称 | 文本 | |
| 专业代码 | 文本 | |
| 所属门类 | 单选 | |
| 内容状态 | 单选 | 草稿/审核中/已发布 |
| 基础信息 | 文本 | JSON或长文本 |
| 过来人说 | 文本 | 关联专家 |
| 更新时间 | 修改时间 | |
| 负责人 | 人员 | |

### 2.3 创建表单

#### 学生提问表单

1. 在「问答管理表」中点击「表单」视图
2. 设置表单标题：「我要提问」
3. 选择需要填写的字段：
   - 问题内容（必填）
   - 问题分类（必填）
   - 相关专业
   - 提问者联系方式（选填）
4. 设置表单样式：
   - 上传Logo
   - 设置主题色
5. 发布表单：
   - 复制表单链接
   - 嵌入到Notion页面

#### 专家入驻表单

1. 在「专家信息表」中点击「表单」视图
2. 设置表单标题：「专家入驻申请」
3. 选择需要填写的字段
4. 添加说明文字：
   - 平台介绍
   - 认证说明
   - 隐私保护承诺
5. 发布表单

### 2.4 设置自动化通知

#### 新提问通知专家

1. 在多维表格中点击「自动化」
2. 创建自动化规则：
   - 触发条件：新增记录且问题分类为「XXX专业」
   - 执行动作：发送消息到群聊/个人
3. 按专业创建多个自动化规则

#### 新专家申请通知管理员

1. 创建自动化规则：
   - 触发条件：新增记录
   - 执行动作：发送消息给管理员

### 2.5 创建专家群

1. 为每个专业领域创建专家群
2. 群名称：如「专业真相-计算机专家群」
3. 设置群公告：
   - 平台介绍
   - 答题指南
   - 福利说明
4. 配置机器人：
   - 新问题自动推送到群
   - @相关专家提醒

## 第三部分：微信公众号配置

### 3.1 注册公众号

1. 访问 [微信公众平台](https://mp.weixin.qq.com)
2. 注册账号：
   - 类型：订阅号（个人可注册）
   - 名称：专业真相
   - 头像：Logo
3. 完成认证（服务号需要，订阅号可选）

### 3.2 配置菜单

在「自定义菜单」设置：

```
一级菜单：专业百科
└── 跳转网页：Notion站点链接

一级菜单：我要提问
└── 跳转网页：飞书表单链接

一级菜单：关于我们
├── 平台介绍
├── 成为专家 → 飞书表单链接
└── 联系我们
```

### 3.3 设置自动回复

**被关注回复**：
```
欢迎来到「专业真相」！

🎓 我们是来自各行各业的过来人
💡 免费分享真实的专业经历
❤️ 帮助学弟学妹做出更好的选择

👇 点击下方菜单开始探索

有任何问题，随时给我们留言！
```

**关键词回复**：
- 「提问」→ 提问表单链接
- 「专家」→ 专家入驻表单链接
- 「专业」→ 专业百科链接

## 第四部分：数据统计配置

### 4.1 飞书仪表盘

1. 在多维表格中创建「仪表盘」视图
2. 添加统计组件：
   - 专家总数
   - 今日新增提问
   - 待回答问题数
   - 问答总数

### 4.2 用户行为分析（可选）

使用「百度统计」或「友盟」：
1. 注册账号
2. 创建站点
3. 获取统计代码
4. 在Super.so后台添加代码

## 第五部分：内容运营流程

### 5.1 日常运营流程

```
每日：
├── 查看待回答问题
├── 分配问题给专家
├── 审核专家入驻申请
└── 回复用户留言

每周：
├── 整理优质问答
├── 更新专业百科
├── 发布公众号文章
└── 专家社群互动

每月：
├── 数据统计分析
├── 专家激励发放
├── 内容质量审核
└── 用户反馈处理
```

### 5.2 内容更新流程

1. 收集专家贡献的内容
2. 按模板整理内容
3. 发布到Notion页面
4. 更新飞书数据表状态
5. 通知相关专家
