## 二、申请条件

### ✅ 基本条件

- 毕业满2年，或工作满1年
- 所从事工作与所学专业相关
- 愿意真实、客观地分享经验

### ✅ 认证材料

请准备以下材料（任选其一）：

| 材料类型 | 要求 |
|---------|------|
| 学历证明 | 学位证书照片 或 学信网截图 |
| 工作证明 | 工牌照片 或 在职证明 或 名片 |
| 补充材料 | LinkedIn主页 或 作品集链接 |

## 四、填写申请表

### 基本信息

| 字段 | 说明 |
|------|------|
| 姓名 | 真实姓名（仅平台可见） |
| 昵称 | 对外展示的昵称 |
| 当前公司 | 如不便透露可写「某互联网公司」 |
| 职位 | 具体职位名称 |
| 工作年限 | 从事该行业年限 |
| 毕业院校 | 本科/研究生院校 |
| 所学专业 | 擅长的专业领域 |
| 学历 | 本科/硕士/博士 |

### 个人简介（示例）

```

某大厂后端工程师，工作5年。本科就读于某985计算机专业。
经历过校招、社招、跳槽，对这个行业有比较深的理解。
希望分享真实的经验，帮助学弟学妹少走弯路。

示例2：
某国企财务主管，工作8年。会计学专业毕业。
从出纳做起，经历过审计、财务分析、预算管理等多个岗位。
想告诉学弟学妹会计学真实的工作状态和发展路径。
```

## 五、回答问题指南

### ✅ 什么样的回答是好的回答？

**真实**
- 基于自己的真实经历
- 不夸大、不美化、不恐吓
- 可以说「不确定」，不要编造

**具体**
- 给出具体例子和数据
- 不要只说「还行」「一般」
- 「我当时的工资是...」「我们公司招人时...」

**中立**
- 呈现多面性，不只有好或只有坏
- 说明「这取决于...」
- 不引导选择特定学校或机构

**有建设性**
- 给出可操作的建议
- 如果是负面信息，说明如何规避或应对
- 用「如果重来我会...」的视角

### ❌ 什么样的回答是不好的回答？

**避免**：
- 虚构经历
- 情绪化吐槽
- 推销任何产品/机构
- 涉及个人隐私（自己的或公司的）
- 泄露商业机密
- 过于主观的判断（「这专业就是垃圾」）

### 回答模板

```
【背景】关于这个问题，我的情况是...

【事实】实际情况是这样的...

【建议】基于我的经验，建议你...

【补充】还有一些注意事项...
```

## 六、专家等级体系

### 等级说明

| 等级 | 图标 | 条件 | 特权 |
|------|------|------|------|
| Lv1 铜牌 | 🥉 | 已认证 | 回答问题、展示主页 |
| Lv2 银牌 | 🥈 | 回答>10个 + 好评率>80% | 优先展示、专属标识 |
| Lv3 金牌 | 🥇 | 回答>50个 + 好评率>90% | 推荐位、年度认证 |
| Lv4 钻石 | 💎 | 平台特邀 | 官方合作机会 |

### 如何升级？

**从铜牌到银牌**：
- 回答至少10个问题
- 获得至少80%的好评率
- 活跃时间超过30天

**从银牌到金牌**：
- 回答至少50个问题
- 获得至少90%的好评率
- 被收藏超过100次
- 活跃时间超过90天

**金牌到钻石**：
- 需要平台主动邀请
- 持续高质量贡献
- 在领域内有影响力

## 七、常见问题

### Q: 我的工作和大学专业不一样，可以申请吗？
> A: 可以。请在申请时说明你的专业背景和职业转型经历，这对想转行的学生很有价值。

### Q: 我可以匿名吗？
> A: 你的真实姓名仅平台可见，对外展示可以用昵称。公司名称也可以用「某互联网公司」等模糊表述。

### Q: 我需要回答多少问题？
> A: 没有硬性要求。我们理解大家工作忙，哪怕只回答1-2个问题也很有价值。

### Q: 回答问题有时间限制吗？
> A: 没有。你可以在方便的时候回答，系统会通知你待回答的问题。

### Q: 我可以获得报酬吗？
> A: 目前平台是公益性质，专家不以回答获得报酬。但你可以获得：
> - 个人品牌曝光
> - 认证证书
> - 行业人脉
> - 如果开通付费咨询，可以获得分成（后续功能）

### Q: 我可以退出吗？
> A: 随时可以申请退出，你的个人信息将被删除，回答内容可以选择保留或删除。
